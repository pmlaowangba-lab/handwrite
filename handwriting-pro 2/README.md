# Handwriting Pro 2（React 前端）

当前版本已接入 Phase 1 后端渲染任务 API：

- 导出：提交后端任务并下载结果
- 打印：提交后端任务并打印结果

## Run Locally

**Prerequisites:** Node.js，且后端服务可用（默认 `http://127.0.0.1:9000`）


1. Install dependencies:
   `npm install`
2. （可选）在 `.env.local` 配置后端地址：
   `VITE_BACKEND_BASE_URL=http://127.0.0.1:9000`
3. Run the app:
   `npm run dev`
